# Gym Data Mining Project

## Overview

This project focuses on analyzing gym members' data using data mining techniques.
The goal is to understand user behavior, build predictive models, and extract useful insights.

This work follows the CRISP-DM methodology as part of a Data Mining course project.

---

## Domain

Fitness Analytics

The project aims to help gym management understand members’ performance and behavior.

---

## Objectives

* Understand gym members' characteristics
* Explore factors affecting calories burned
* Prepare data for future modeling
* Plan for clustering, classification, and regression tasks

---

## Dataset

* Source: Kaggle
* Link: https://www.kaggle.com/datasets/valakhorasani/gym-members-exercise-dataset

### Dataset Summary

* ~974 records
* 15 features
* Contains numerical and categorical data
